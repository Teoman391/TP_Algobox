Ce projet a été réalisé dans le cadre d’un travail collaboratif visant à découvrir l’utilisation de GitHub et Git Bash tout en développant un algorithme complet sous AlgoBox.
L’objectif du programme est de calculer la moyenne d’une classe composée d’un nombre variable d’élèves, tout en s’assurant que chaque saisie de note soit correcte.
Le programme affiche également la note minimale et la note maximale obtenues dans la classe.

Saisie du nombre d’élèves

  - Vérification de la validité des notes (comprises entre 0 et 20)

  - Calcul automatique :

      - de la moyenne,

      - de la note minimale,

      - de la note maximale

  - Utilisation d’une boucle TANT QUE pour garantir une saisie correcte

POUR L'UTILISER

Cloner le dépôt :

  - git clone https://github.com/Teoman391/TP_Algobox.git

  - Ouvrir le fichier .alg dans AlgoBox.

  - Exécuter l’algorithme et suivre les instructions à l’écran.
